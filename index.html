How to use

Set up your cones

Place 4 cones on the ground matching the grid on screen — front left, front right, back left, back right. Stand in the middle.

Configure your drill

  • Movement pattern — how you travel to each cone (sprint, shuffle, backpedal, crossover, or random)
  • Time between calls — how long you have to reach each cone before the next one is called
  • Reps per cone — how many times each cone gets called before the set ends

Run the drill

Tap Start. A 5-second countdown plays, then the app calls a cone name and movement out loud. Sprint to that cone, return to the middle, and wait for the next call.

Cones turn green when active and blue when their reps are done. The set ends automatically when all cones are complete.

Pause / Reset

Tap Pause mid-set to stop the timer. Tap Reset to clear everything and start fresh.
